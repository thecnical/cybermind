# CyberMind AI Fine-Tuning

Fine-tune Llama 3.2 3B into a world-class bug bounty AI — **free**, using Google Colab.

## What You Get

A fully uncensored, security-focused AI that:
- Thinks like a top 1% bug bounty hunter
- Knows every attack technique, CVE, and bypass method
- Makes autonomous decisions in the agentic pipeline
- Provides exact commands, payloads, and PoCs

## Datasets Used

| Dataset | Source | Size | Purpose |
|---------|--------|------|---------|
| Bug Hunter | Kaggle: `vellyy/bug-hunter` | ~50MB | Real bug bounty reports |
| CVE Dataset | Figshare DOI: 10.6084/m9.figshare.22056617 | ~200MB | CVE knowledge base |
| CyberMind Synthetic | Generated by this repo | Dynamic | Agentic behavior |

## Quick Start

1. Open `CyberMind_FineTune.ipynb` in Google Colab
2. Set Runtime → T4 GPU (free)
3. Run all cells
4. Download the model
5. Use with CyberMind: `cybermind /groq-setup` or Ollama

## How to Use After Training

```bash
# Option 1: Ollama (local, free, slow on CPU)
ollama create cybermind-security -f Modelfile
OLLAMA_MODEL=cybermind-security cybermind /plan example.com --local

# Option 2: Groq (cloud, free, fast)
cybermind /groq-setup gsk_YOUR_KEY_HERE

# Option 3: Backend (for all users)
# Add to Render env: GROQ_KEYS=gsk_key1,gsk_key2
```
