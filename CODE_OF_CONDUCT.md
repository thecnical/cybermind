# Code of Conduct

## Our Pledge

CyberMind is a professional cybersecurity research tool. All contributors and users are expected to:

- Be respectful and professional
- Use the tool for legal, authorized security research only
- Not use CyberMind for illegal activities, unauthorized access, or harm

## Responsible Use

CyberMind provides cybersecurity knowledge for:
- Authorized penetration testing
- CTF competitions
- Security research and education
- Bug bounty programs

**Never use CyberMind against systems you do not own or have explicit written permission to test.**

## Enforcement

Violations of this code of conduct may result in removal from the project.
Report issues to: github.com/thecnical
