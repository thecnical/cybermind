# CyberMind

AI-powered cybersecurity CLI tool.

## Tech Stack

- Go (CLI)
- Node.js (Backend)

## Structure

```
CyberMind/
├── backend/   # Node.js + Express API
└── cli/       # Go CLI client
```

## Quick Start

Backend:
```bash
cd backend
npm install
node src/app.js
```

CLI:
```bash
cd cli
go run main.go
```
